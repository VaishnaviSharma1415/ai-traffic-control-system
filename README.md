# AI Traffic Control System 🚦

This project uses YOLOv8 and Streamlit to detect vehicles and optimize traffic signal timing.

## Features
- Vehicle detection (car, bike, bus, truck)
- Real-time traffic analysis
- Dynamic green signal timing

## Technologies Used
- Python
- YOLOv8
- Streamlit
- OpenCV

## How to Run
```bash
pip install -r requirements.txt
streamlit run ai.py
